# 🌟 StellarYield: Decentralized Micro-Lending on Stellar

**Live Demo**: [https://ais-dev-ahjekcwqaqztnudwoelinh-60290472169.asia-southeast1.run.app](https://ais-dev-ahjekcwqaqztnudwoelinh-60290472169.asia-southeast1.run.app)  
**Demo Video**: [Link to Video Placeholder]

StellarYield is a native Stellar DeFi protocol that provides algorithmic interest rates based on your on-chain reputation. By bypassing traditional collateral requirements for high-reputation accounts, we enable true micro-lending for the Stellar ecosystem.

## 🚀 Key Features

- **Decentralized Reputation**: No credit score needed. We use your Stellar transaction history.
- **Algorithmic Yield**: Shared liquidity pools for XLM and USDC.
- **Non-Custodial**: Integrated with Freighter Wallet.
- **Soroban Powered**: Scalable and secure smart contract logic.

## 🛠 Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion.
- **Blockchain**: Stellar JS SDK, @stellar/freighter-api.
- **API**: Stellar Horizon (Testnet).
